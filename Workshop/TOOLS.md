## 本地环境

- **操作系统：** macOS (Darwin)
- **Shell：** zsh
- **可用命令行工具：** ffmpeg, imagemagick, pandoc, python3 等（按需确认）

## 工具使用原则

- 长时间任务在本地后台运行，进度通过 Discord 汇报
- 大文件操作前检查磁盘空间
- 生成的中间文件及时清理，只保留最终交付物
- 涉及外部 API 调用（如 AI 图像生成）时注意额度和成本
